# Random User Generator - CRUD API

Projeto desenvolvido como desafio técnico para **Paschoalotto**.  
Consiste em uma API em **ASP.NET Core + PostgreSQL** integrada à API pública [Random User Generator](https://randomuser.me/),  
com um **frontend simples em HTML + JS** para consumir os dados.

---

## 🚀 Tecnologias
- C# / .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- JavaScript (fetch API)
- HTML + CSS
- Swagger para documentação

---

## ⚙️ Funcionalidades
- ✅ Consumir usuários aleatórios da API [randomuser.me](https://randomuser.me)
- ✅ Salvar usuários no banco PostgreSQL
- ✅ Listar todos os usuários
- ✅ Buscar usuário por ID
- ✅ Editar usuários
- ✅ Excluir usuários
- ✅ Frontend básico para CRUD
